# S2 좁은 클리닝 패치 영향 리포트

작성일: 2026-05-12

## 목적

`[효종]` 전체 대괄호 래핑 복구와 `파공검제 30권`류의 말미 `제` 보존 패치가 실제 검토 대상에 미치는 영향을 확인한다.

## 검증 방식

- 입력: `텍스트 클리닝 고도화.xlsx`
- S2 기준: `data/kiss_payment_settlement_s2_lookup.csv`
- 패치 전 `clean_title` 로직을 스크립트 안에 복원했다.
- 정산서 원문과 S2 콘텐츠명을 각각 패치 전/후 로직으로 다시 정제했다.
- `판매채널 + 정제키` 기준으로 S2 매칭 상태를 비교했다.

저장된 엑셀의 `정제_상품명` 컬럼을 그대로 비교하지 않고, 원문에서 패치 전/후 정제키를 재계산했다. 이 방식은 엑셀의 `24/7` 날짜 자동 변환 같은 표시 잡음을 피하기 위한 것이다.

## 전체 영향

| 항목 | 패치 전 | 패치 후 | 증감 |
| --- | ---: | ---: | ---: |
| matched | 75,089 | 75,997 | +908 |
| no_match | 7,867 | 6,960 | -907 |
| blank_key | 1 | 0 | -1 |

상태가 바뀐 행은 총 908행이며, 전부 `no_match/blank_key -> matched` 방향이다.

## 판정

이번 패치는 의도한 5개 그룹만 움직였다.

- 기존 matched가 no_match로 떨어지는 회귀는 발견되지 않았다.
- 변경 행은 모두 S2 같은 판매채널 후보가 있는 행이다.
- `귀축기담` 같은 “다른 채널에는 있으나 해당 판매채널에는 없는” 케이스는 이번 패치로 움직이지 않는다.
